# Sistema de Biblioteca Acadêmica (Java)

Este repositório contém o código-fonte de um sistema de gerenciamento para uma Biblioteca Acadêmica, desenvolvido inteiramente em Java.

## Objetivo do Projeto
O objetivo principal foi aplicar na prática os fundamentos de Programação Orientada a Objetos (POO) na construção da lógica de back-end. O projeto foca na estruturação correta do código, na separação de responsabilidades e na manipulação de dados em memória.

## Tecnologias e Conceitos Aplicados
* **Linguagem:** Java
* **IDE Utilizada:** NetBeans
* **Paradigma:** Programação Orientada a Objetos (POO)
* **Fundamentos Explorados:**
  * Modelagem de Classes e Objetos.
  * Uso de construtores para inicialização e validação de estado.
  * Encapsulamento de dados.
  * Gerenciamento dinâmico do acervo utilizando listas de publicações.

## Como Executar
O projeto pode ser clonado e executado em qualquer IDE com suporte a Java (NetBeans, Eclipse, IntelliJ). Basta compilar e rodar a classe principal para interagir com o sistema.
